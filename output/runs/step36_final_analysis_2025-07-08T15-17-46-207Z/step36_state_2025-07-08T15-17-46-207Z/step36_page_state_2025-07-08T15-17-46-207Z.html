Firma de documentos con Token

Instalar extensión de navegador Chrome

No se encuentra la extensión de Chrome requerida para firmar con token.

Para continuar con el proceso de firma deberá instalarla.

El proceso consta de 2 pasos: la instalación de la extensión y de una aplicación, que una vez descargada, deberá ejecutarla.

Una vez completados ambos pasos, deberá volver a retomar el trámite y continuar con la firma del documento.

Firma de documentos con Token

Instalar extensión de navegador Chrome (paso 2)

Ya se encuentra instalada la extensión pero se requiere la instalación de otra aplicación (paso 2).

Para continuar presione el botón "Descargar" y una vez descargado el archivo .msi, debe ejecutarlo para completar el proceso.
